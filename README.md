# Green-phoenix

Green-Phoenix is a personal project designed to put patients and doctors in touch with each other. 

## Technology used

Back-end
- Symfony 6
- Webpack Encore
- Sass
- Git Versioning
- FullCalendar

Front-end
- Bootstrap

Database
- MySQL

## Features
- Doctor side
  - Implementation of full calendar ( Crud of Events)
 
- Patient Side
  - Search a doctor
  - Take an appointment with a doctor
  - View all appointments
- General
  - Authentification
  - API
